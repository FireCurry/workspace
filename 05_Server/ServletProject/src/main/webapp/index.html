서버 제출 테스트 1

    - 이름과 나이를 입력 받아 서버로 제출

    * 준비물 *
    1) input 태그 : 값을 입력하는 양식을 제공
      - name 속성 : 서버로 제출 시 Key 역할을 하는 속성
      - input에 입력된 값 또는 value 속성의 값 : Value 역할
    
    2) form 태그 : 내부에 작성된 input 값을 지정된 경로로 제출
      - 지정된 경로로 제출 == 클라이언트가 서버에 요청
      - action 속성 : 제출할 경로(주소)
                      (주소 == 무슨 요청인지 지정)
      - method 속성 : 제출 방법(데이터 전달 방식)
  
      - action="/example1"
        -> 최상위주소/example1 주소로 요청을 보내겠다(제출하겠다)

      - method="GET"
        -> 제출되는 데이터를 주소에 담아서 제출하겠다
          ex) 주소?name=홍길동&age=20
                  (쿼리 스트링)
