하이퍼 링크 관련 태그 : a(anchor, 연결, 고정) 태그

    a태그는 현재 HTML 문서와 다른 HTML 문서를 연결해서
    페이지 이동을 할 수 있게 하는 기능을 제공
    -> 이 때, a태그로 감싸진 문자열 == 하이퍼링크(또는 하이퍼 텍스트)
    * 하이퍼링크는 웹(html) 문서가 다른 문서와 구분되는 핵심적인 기능

a태그를 이용한 하이퍼 링크 구현

    href(HyperText Reference) 속성 : 연결할 주소를 지정하는 속성
    target="_blank" : 새 탭에서 열기
